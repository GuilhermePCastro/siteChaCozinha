Como funciona nossa lista de presentes?

Para tornar este processo ainda mais especial e compartilhar cada passo com vocês, criamos esta lista de presentes virtual. Aqui, explicamos de forma simples e carinhosa como você pode fazer parte deste sonho:

1
Escolha o presente

Navegue pela nossa lista, escolha o item e saiba que o valor será revertido para nós.

2
Coloque seu nome

Preencha seu nome para sabermos quem nos enviou este gesto de carinho.

3
Escreva uma mensagem

Deixe um recado especial que guardaremos com muito carinho.

4
Pagamento via PIX

Um QR Code será gerado para o pagamento. É só escanear com o app do seu banco.

5
Confirme o envio

Após pagar, clique em "Confirmar Presente" para nos notificar. Pronto!
